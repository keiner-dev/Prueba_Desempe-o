# The Beatles - Rock Landing Page

## Description
This is a professional landing page for a rock band designed to showcase their identity, upcoming tour dates, and multimedia content. The project focuses on a "Rock & Roll" aesthetic with a dark theme, high contrast, and responsive design.

## Technologies Used
- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Modern layout techniques including Flexbox, CSS Grid, and Media Queries.
- **JavaScript**: Simple interactions for theme toggling and dynamic content.

## Project Structure
- `index.html`: Main structure of the site.
- `styles.css`: Visual styling and responsiveness.
- `script.js`: Logic for UI interactions.
- `assets/`: Directory for band images and icons.

## Features
- Fully responsive layout (Mobile, Tablet, Desktop).
- Dynamic events table with ticket simulation.
- Theme Switcher (Dark/Light mode) using CSS variables.
- Scroll Reveal animations for better UX.
- Musical influences list and social media integration.
